What is the Document Object Model (DOM)?

Document Object Model or DOM is an Api which represents the structure of a html Document. It makes all the element in object type in tree shape to mimic the hierarchy of a webpage. With the help of DOM in javascript we can access and change an element and modify their property to create a dynamic webpage.
